YouTubePlayer.OnFullscreenListener

com.google.android.youtube.player.YouTubePlayer.OnFullscreenListener

Class Overview

Interface definition for callbacks that are invoked when the player toggles between fullscreen on or off, either due to the user clicking the fullscreen button or a call to setFullscreen(boolean).
